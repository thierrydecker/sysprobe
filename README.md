## SysProbe

A Python system monitor agent

### Introduction

SysProbe collects system informations and logs them to a InfluxDB database.

### Tracked counters

- CPU counters
- Memory counters
- Disks counters
- Processes counters
- Network counters

### Developped on

- Windows 10
- Python v3.6
- InfluxDB v1.4

### Main dependencies

- PyYAML v3.12
- schedule v0.5.0
- psutil v5.4.3
- influxdb (Python client) v5.0.0
